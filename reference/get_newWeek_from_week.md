# Map surveillance week to newWeek index in a season

Map surveillance week to newWeek index in a season

## Usage

``` r
get_newWeek_from_week(season_df, week_vec)
```

## Arguments

- season_df:

  data frame with columns \`week\` and \`newWeek\`

- week_vec:

  vector of surveillance weeks to map

## Value

integer vector of newWeek indices
